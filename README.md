# tictactoe-project

Project Description

This is a 2-player Tic-Tac-Toe game that runs in the terminal. Players take turns entering row and column positions (0–2) until one player wins or the game ends in a draw.

How to Run

1. Make sure Python is installed

How to Play

* Player X goes first, followed by Player O
* Enter row and column numbers (0–2) when prompted
* You cannot choose an occupied cell
* The game ends when:

  * A player gets 3 in a row (win), or
  * The board is full (draw)


Notes

* Includes input validation
* Tracks score across multiple rounds
* Allows replay
